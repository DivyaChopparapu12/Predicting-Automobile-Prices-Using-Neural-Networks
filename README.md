# Predicting-Automobile-Prices-Using-Neural-Networks
This capstone project investigates the determinants of used-car prices and evaluates which predictive modeling technique - linear regression, decision tree, or neural network, produces the most accurate and reliable price predictions.

## 🔍 Problem
Which vehicle characteristics most influence used-car prices, and which predictive model provides the most reliable results?
This question is highly relevant for used-car dealers seeking data-driven pricing strategies.

## 🛠️ Tools Used
R | Power BI | Linear Regression | Decision Tree | Neural Network

## 📊 Approach
- Analyzed 1,367 real vehicle records
- Built and compared 3 predictive models - linear regression, decision tree, and neural network
- Evaluated performance using RMSE, MAE, and R²

## ✅ Results
Achieved ~90% accuracy with Linear Regression model
Visualized pricing trends in Power BI dashboards
